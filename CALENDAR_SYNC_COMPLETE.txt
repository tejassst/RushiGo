## 🎯 API Endpoints

### New/Enhanced Endpoints
```
POST /calendar/enable
├─ ENHANCED: Now syncs existing deadlines on first enable
├─ Returns: message, calendar_id, synced_count, sync_errors
└─ Smart detection: Only bulk syncs if transitioning from disabled → enabled

PUT /users/me/calendar-preferences  
├─ ENHANCED: Now syncs existing deadlines when enabling
├─ Returns: message, synced_count, sync_errors

### Existing Endpoints (Verified Working)
```
POST /deadlines/create     → Auto-syncs if enabled
PUT /deadlines/{id}        → Auto-syncs changes if synced
DELETE /deadlines/{id}     → Auto-removes from calendar if synced
GET /calendar/status       → Check sync status
POST /calendar/connect     → OAuth flow
POST /calendar/disconnect  → Remove tokens
POST /calendar/sync-all    → Manual bulk sync
